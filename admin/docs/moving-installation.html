Moving Your ToplistX Installation

Table of Contents
Introduction

Use the instructions in this document if you need to move your ToplistX installation to a different server or to a new domain on the same server. Following these instructions will allow you to save all of your current data and will ensure that the new installation will function properly.

Moving Your Installation

Follow these steps to move your ToplistX installation:
  1. Use the instructions in the Database Tools section of this manual to create a backup of your ToplistX database.

  2. Install a new copy of ToplistX on the new server/domain using the normal installation instructions.

  3. Use the instructions in the Database Tools section of this manual to restore the database in your new installation using the backup file created in step 1.

  4. Copy the files from the templates directory (but not any of it's sub-directories) of the old installation to the templates directory of the new installation to restore your templates.
