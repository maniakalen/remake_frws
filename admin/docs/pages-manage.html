Ranking Pages

Table of Contents
Introduction

ToplistX allows you to create an unlimited number of ranking pages to display your accounts. This document will describe how you can add, update, and remove the ranking pages that ToplistX will be building.

While ToplistX does not set a limit on the number of ranking pages you can configure, your server configuration will be a limiting factor. If you plan on having a very large number of ranking pages generated by ToplistX (500+) it is highly recommended that you have a high-end dedicated server.

Searching/Viewing Ranking Pages

To view the currently configured ranking pages so you can modify or delete them, access the ranking Pages > Manage Pages function from the control panel menu. This will display a page where you will be able to see all of the currently configured ranking pages and search them if necessary.

In the table that displays the ranking pages you have configured, you will find several functions that you can run on a single ranking page at a time. Each of those functions are described below.
    Click this icon to go directly to the ranking page template editing interface for this page
    Click this icon to edit the ranking page settings. A window will appear where you will be able to edit all of the page settings.
    Click this icon to delete this ranking page. This will delete the ranking page from the database so it is no longer updated, but will not remove the actual generated ranking page from your server.
In addition to running these functions on single ranking pages, you can also delete multiple pages at once. To do this, put a check in the checkbox next to the ranking pages you want to delete, then press the Delete button at the bottom of the page.

Adding Ranking Pages

To add a new ranking page, access the Ranking Pages > Manage Pages function from the control panel menu. Near the top of the page in the header bar that reads "Manage Ranking Pages" click on the icon. This will display a popup window where you will be able to enter the information for the new page. Each of the form fields are described below.
    Page URL Enter the URL that you would like to use for the new page. The base URL of your site will already be filled in, so you only need to fill in the remaining part of the URL. If this points to a directory that does not yet exist, you will need to create the directory and set it's permissions to 777. If this points to a file that already exists, the file should be deleted or have it's permissions changed to 666; note that the existing file will be overwritten if you simply change it's permissions and do not rename it.
    Category Select a specific category for this page, or MIXED if you want to allow accounts from all categories. If you do select a specific category, only accounts from that category will be allowed to appear on the page
    Build Order Enter a number to indicate the order in which this page will be built. This value will automatically be filled in with the next available position in the build order, but you can change it to any value. Lower numbers indicate pages that will be built first, and larger numbers indicate pages that will be built last.
    Tags You can use this field to specify tags for this page. Tags are useful for selecting only specific pages when you are generating links between your pages. Tags must be at least 4 characters long and should be listed separated by spaces.
Once you have filled in all of the fields as you want them, press the Add Ranking Page button. This will add the new page to the database, and it will appear in the Manage Pages interface. The actual ranking page will not yet be created, that will happen after you run one of the page building functions.

Adding Ranking Pages in Bulk

To add multiple new ranking pages at one time, access the Ranking Pages > Manage Pages function from the control panel menu. Near the top of the page in the header bar that reads "Manage Ranking Pages" click on the icon. This will display a popup window where you will be able to enter the information for the new pages. Each of the form fields are described below.
    Base URL Enter a URL that points to the directory on your server where you want the ranking pages created. This directory should have 777 permissions.
    Category Select a specific category to generate pages for, MIXED if you want to generate general pages that can contain any categories, or ALL CATEGORIES if you want to generate pages for all of your defined categories.
    Filename Prefix If you have selected MIXED for the category, you will need to fill in the filename prefix. This will be the name that will be used to generate the names that are used for the ranking pages. Examples are listed below.
    File Extension Enter the file extension - without a period - to use for the generated pages
    Tags You can use this field to specify tags for the generated pages
    Number of Pages Enter a number to indicate how many pages should be generated. Examples are listed below.
    Conversions Select what conversions should be done to category names when generating the ranking page filenames
Once you have filled in all of the fields as you want them, press the Add ranking Page button. This will add the new page to the database, and it will appear in the Manage Pages interface. The actual ranking page will not yet be created, that will happen after you run one of the page building functions.

Editing a Ranking Page

To edit a ranking page you will first need to locate it through the Manage Pages interface. When you locate the page you want to edit, click on the icon to bring up the page editing interface. This interface is nearly identical to the page creation interface, and has the same form fields with the same properties. You can make the changes that you want, and then press the Update Ranking Page button to save your changes.

Deleting a ranking Page

To delete a ranking page you will first need to locate it through the Manage Pages interface. When you locate the page you want to delete, click on the icon, which will ask you to confirm the deletion.

You can also delete several pages at once by putting a check in the box next to the pages you want to delete, then pressing the Delete button at the bottom of the page.
