Categories

Table of Contents
Introduction

Categories will allow you to split sites up into their own niche sections, and rank the sites against other sites of the same niche. It is not required that you setup categories for ToplistX, but it will give you the ability to have more ranking lists created. This document describes how to manage your categories.

Viewing/Searching Categories

To view or search your currently configured categories, access the Settings > Manage Categories function from the control panel menu. This will display a page where you will be able to see the currently configured categories and search them if you wish.

In the table that displays the categories, you will find a couple of functions that you can run on a single category. Each of those functions are described below.
    Click this icon to view the accounts in this category
    Click this icon to edit the category. A window will appear where you will be able to edit all of it's settings.
    Click this icon to delete this category.
In addition to running these functions on single categories, you can also run the delete function on multiple categories at once. To run the delete function, put a check in the checkbox next to the categories you want to delete, then press the Delete button at the bottom of the page.

The search results will display a few different pieces of information about the category. Each column of the search results is described below.
    Name The name of the category
    Hidden This category is hidden and cannot be selected by account submitters
    This category is not hidden and can be selected by account submitters
    Accounts The number of accounts in this category
    Sorter The value corresponding to the "Sort:" field that you selected for the current search
Adding a Category

To add a new category access the Settings > Manage Categories function from the control panel menu. This will display your currently configured categories. Near the top of the page in the header bar that reads "Manage Categories" click on the icon. This will display a popup window where you will be able to enter the data for the new category that you want to add. Each of the form fields are described below.
    General Settings
    Category Name(s) The name(s) you want to use for the category(ies). You can enter one category per line to add multiple categories at once.
    Forward URL If you are using the "Unique link for each account" tracking mode, this is the URL surfers will be sent to when they come from a site in this category
    Page URL This value will be assigned to the {$account.category_url} for all accounts in this category
    Make this category hidden
    If this is checked this category will not appear on the end-user account submission form and users will not be able to submit accounts to this category
    Account Settings
    Site Title Length The number of characters allowed in the site titles of accounts in this category
    Description Length The number of characters allowed in the site titles of accounts in this category
    Allow redirecting site URLs to be submitted (300 level HTTP status codes)
    Check this box to allow the site URL that is being submitted to redirect surfers to a different URL
    Account Banner Settings
    Max Banner Width The maximum banner width (in pixels) allowed in this category
    Max Banner Height The maximum banner height (in pixels) allowed in this category
    Max Banner Filesize The maximum banner file size (in bytes) allowed in this category
    Force all banners to the height and width entered above
    Check this box to force all banners to have the dimensions entered above. Smaller banners will be stretched to this size.
    Download banners to check height, width, and filesize
    Check this box to have the banner dimensions and filesize checked during account submission. This will download the banner from the remote site to check this information, but will not permanently store the banner on your server.
    Host member account banners from your server
    Check this box to have all member banners served from your server. This will download the banner from the remote site and store it permanently on your server so when people access your ranking list the banners will load from your server.
Once you have filled in all of the fields as you want them, press the Add Category button. This will add the new category to the database, and it will be activated and ready for accounts immediately.

Editing a Category

To edit a category you will first need to locate it through the View/Search interface. When you locate the category you want to edit, click on the icon to bring up the category editing interface. This interface is nearly identical to the category addition interface, and has the same form fields with the same properties. You can make the changes that you want, and then press the Update Category button to save your changes.

Deleting a Category

Deleting a category will also delete all of the accounts contained in that category!

To delete a category you will first need to locate it through the View/Search interface. When you locate the category you want to delete, click on the icon, which will ask you to confirm the deletion.

You can also delete several categories at once by putting a check in the box next to the categories you want to delete, then pressing the Delete button at the bottom of the page.
