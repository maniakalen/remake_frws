Link Searching

Introduction

The link search function in LinkX allows you to quickly locate links so they can be approved, edited, deleted, etc. This document provides an overview of the link search interface.

Searching Links

To access the link search interface use the Links > Search Links function from the control panel menu. This will load a page which will initially indicate that no search term has been entered. To start your search, enter the term you would like to search for in the text input box and select the field that you would like to search in. Each of the search fields is described below.

The Title, Description or Keywords field supports boolean mode searching; the other fields support standard searching.
    Title, Description or Keywords This will search in the link title, description, and keywords and will return results that match in any of those fields.
    Link ID The ID number that was assigned to the link when it was added
    Site URL The full URL of the site that surfers are sent to when they click on this site link
    Recip URL The full URL of the page where the site owner plans to place a link back to your site
    Title The site title that was provided when the link was created or last updated
    Description The site description that was provided when the link was created or last updated
    Status The current link status (one of unconfirmed, pending, active, or disabled)
    Type The current link type (one of regular, premium, or featured)
    Expires The date that the link becomes a regular link (must be in YYYY-MM-DD format; applies to links that currently have premium or featured status)
    Name The name that was provided when the link was created or last updated
    E-mail Address The e-mail address that was provided when the link was created or last updated
    Submit IP The IP address that was used when the link was created or last updated
    Keywords The keywords that were used when the link was created or last updated
    Clicks The number of clicks this link has received
    Number of Ratings The number of ratings this link has received
    Average Rating The average rating this link has received
    Weight The weight value assigned to this link
    Date Added The date this link was added (must be in YYYY-MM-DD HH:MM:SS format)
    Date Modified The date this link was last modified (must be in YYYY-MM-DD HH:MM:SS format)
    Date Scanned The date this link was last scanned (must be in YYYY-MM-DD HH:MM:SS format)
    Icon HTML The icon HTML code assigned to this link
    Admin Comments The admin comments assigned to this link
    Username The username of the account that this link was submitted by
In addition to searching the fields above, you can also select to display links that have a specific status. Each of the possible status options are described below.
    All This will display all links, no matter what their status
    Unconfirmed This will display links that have not yet been confirmed through the submitter's e-mail address
    Pending This will display links that have been submitted but are awaiting approval by an administrator
    Active This will display links that have been approved and are currently displayed in the link directory
    Disabled This will display links that are disabled and are not visible in the link directory
If you would like to view links that have been edited and are awaiting approval of the changes they have made, check the Edited checkbox. You can also choose how the links are sorted by selecting the sorting field from the drop down list. The available sorting fields correspond with the search fields which are described above. Once you have the search values filled in how you want them, press the search button to display the links that meet your criteria.

Link Functions

In the links section you will be able to run a number of different functions on the links that are displayed. By default the following link information will be displayed:
  • The site title, which is linked to the site URL
  • The site description
  • The link weight value
  • The number of clicks the link has received
  • The number of comments for the link
  • The number of ratings the link has received
  • The link's average rating
To display additional information about the link, click on the icon. This will send a request to your server for the information, so it may take a couple of seconds to load. Once it has been loaded, the information will slide into view. All pending links will have this additional information automatically displayed so you can see it quickly when you are processing new submissions.

Clicking on the icons in the Functions column of the table will allow you to run that function on a single link. You can also check the boxes next to multiple links to perform the functions listed at the bottom of the page. The single link function icons are each described below.
    Click this icon to edit the link. A window will appear where you will be able to edit all of the link settings.
    Click this icon to scan the link. A window will appear with the scan results.
    Click this icon to blacklist the link. A window will appear allowing you to select which items to blacklist.
    Click this icon to view the user account that submitted the link (visible only on links submitted by an account)
    Click this icon to view the comments that have been submitted for this link (visible only on links with comments)
    Click this icon to approve the user modifications to this link (visisble only on edited links)
    Click this icon to reject the user modifications to this link (visisble only on edited links)
    Click this icon to re-activate a disabled link (visible only on disabled links)
    Click this icon to disable an active link (visible only on active links)
    Click this icon to approve or reject a pending link (visible only on pending or unconfirmed links)
    Click this icon to send an e-mail message to the link submitter. A window will appear where you can enter the e-mail text
    Click this icon to delete the link. You will be asked to confirm the deletion before it is done.
You can also run a function on multiple links by putting a check in the box next to the links you want, selecting the function to run from the drop down selection field at the bottom of the page, and pressing the Execute button. Each of the available functions are described below.
    E-mail Selected This function will display a window where you can enter your e-mail text and send an e-mail message to the selected links
    Delete Selected This will permanently delete the links you have selected
    Approve Selected This will approve the links you have selected, allowing you to select the e-mail message to send to the link submitters (this function should be used only for pending or unconfirmed links)
    Reject Selected This will reject the links you have selected, allowing you to select the e-mail message to send to the link submitters (this function should be used only for pending or unconfirmed links)
    Disable Selected This will set the selected links' status to disabled
    Activate Selected This will set the selected links' status to active
    Blacklist Selected This will automatically blacklist the selected links. The site URL, e-mail address, and submit IP will be added to the blacklist
    Accept Edits of Selected This will accept the user modifications of the selectd links (this function should only be used for edited links)
    Reject Edits of Selected This will reject the user modifications of the selectd links (this function should only be used for edited links)
    Change Category of Selected This will display a window where you will be able to select the categories to move the selected links into
You can add a new link by clicking on the icon in the header bar. Clicking on this icon will display a window where you can enter the link details.

Link Quick Tasks

The quick tasks interface for links will allow you to perform search and replace, search and set, and search and delete functions. You can access the quick tasks interface by clicking on the icon in the header bar above the search section. Please see the Link Quick Tasks section of the software manual for additional information.
