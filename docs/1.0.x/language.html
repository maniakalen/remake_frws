Introduction

The ToplistX language file allows you to translate error and other informative messages into your native language. This document describes how to modify the language file settings.

Control Panel Editing Interface

All changes to your language settings will be made through the control panel. To edit your language settings access the Templates > Language File function from the control panel menu. A page will load where you can edit all of the available language settings. Each of the items should be self-explanatory by reading the existing text. You can modify the settings to change the messages or translate them into your language. Once you have made the changes you want, press the Save Language File button at the top of the page.

Value Placeholders

Some of the default messages have value placeholders (%d, %s, etc) embedded in them so that dynamic information can be inserted into them. Each of the language file settings that has these placeholder values are described below.
    CURLE_COULDNT_CONNECT %s will be replaced by a software generated error message
    CURLE_HTTP_RETURNED_ERROR %s will be replaced by a the returned HTTP status
    INVALID_URL %s will be replaced by the name of the improperly formatted URL
    COMMENT_LENGTH The first %d will be the minimum comment length
    The second %d will be the maximum comment length
    DESCRIPTION_LENGTH The first %d will be the minimum description length
    The second %d will be the maximum description length
    TITLE_LENGTH The first %d will be the minimum title length
    The second %d will be the maximum title length
    BLACKLISTED The first %s will be the blacklist item that was matched
    BLACKLISTED_REASON The first %s will be the blacklist item that was matched
    The second %s will be the reason why this item is in the blacklist
    BROKEN_URL The first %s is the type of URL submitted
    The second %s is the error message associated with this URL
    BAD_BANNER_BYTES %d is the maximum banner byte size you allow, taken from the category configuration
    BAD_BANNER_SIZE The first %d is the maximum allowed banner width, taken from the category configuration
    The second %d is the maximum allowed banner height, taken from the category configuration
    MAXIMUM_KEYWORDS %d is the maximum number of keywords you allow, taken from the software configuration settings
    RATING_RANGE %d is the maximum surfer rating you allow
