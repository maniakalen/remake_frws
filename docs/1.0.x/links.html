Links

Introduction

The main function of LinkX is to maintain a database of links, so it is obviously important to understand how the software handles them. This document will give you an overview of how to add, maintain, and remove links.

Adding a New Link

To add a new link access the Directory > Links > Add a Link function from the control panel menu. This will display a popup window where you will be able to enter the link data that you want to use. Each of the form fields are described below.
    E-mail The e-mail address that should be associated with this link
    Submitter Name The name of the link submitter
    Submitter IP The IP address of the link submitter
    Site URL The full URL of the site that surfers are sent to when they click on this site link
    Clicking on the icon will have the software access the site URL and extract the title, meta description, and meta keywords values from the site for use in the corresponding fields on the link submission form.
    Recip URL The full URL of the page where the site owner plans to place a link back to your site
    Title The title of the site
    Description A description of the site
    Status The link status (one of unconfirmed, pending, active, or disabled)
    Type The link type (one of regular, premium, or featured)
    Expires The date that the link becomes a regular link (must be in YYYY-MM-DD format; applies to premium or featured links)
    Keywords Keywords that will be used for search purposes (a space separated list of words)
    Clicks The number of clicks this link has received
    Number of Ratings The number of ratings this link has received
    Ratings/Total The first field is the number of ratings this link has received
    The second field is the total value of all the ratings when added up
    Weight The weight value assigned to this link
    Date Added The date this link was added (must be in YYYY-MM-DD HH:MM:SS format)
    Date Modified The date this link was last modified (must be in YYYY-MM-DD HH:MM:SS format)
    Icon HTML The icon HTML code assigned to this link
    Admin Comments The admin comments assigned to this link
    Account The username of the account that this link was submitted by
    Password The password associated with this link
    Categories Select the categories that this link will appear in. Click the icon to select categories or the icon to clear the current selected categories. You will use the Category Selector interface to select categories.
    Require a reciprocal link Check this box if this site must have a reciprocal link back to your site. This overrides the global and account level settings.
    Allow URL redirection Check this box to allow this site URL to redirect to other URLs. This overrides the global and account level settings.
You will also find all of the user defined fields that you have configured, and will be allow to fill in or select the appropriate values for those fields. Once you have filled in all of the fields as you want them, press the Add Link button. This will add the new link to the database.

Link Status

When links are submitted they are assigned a status which determines if they are displayed in the link directory. Each of the available status options are described below.
    Unconfirmed Links that have been submitted but not yet confirmed through the submitter's e-mail address have this status
    Pending Links that have been submitted but not yet approved by an administrator have this status
    Active Links that have been approved and are currently displayed in the link directory have this status
    Disabled Links that have been disabled by an administrator and are not displayed in the link directory have this status
The link status can be changed at any time by editing a link or by using the functions available in the Directory Browsing or Link Search interfaces

Processing New Links

If you have one or more of your categories setup to require approval of new links, you will need to login to the control panel from time-to-time to review newly submitted links. To view the newly submitted links, select To Do > Review New Links from the control panel menu. This will load the Link Search interface with the appropriate options set to display only links with pending status.

New links can be processed one at a time, or in bulk. When viewing the new links, you will see the and icons added to the Functions column along with a drop down selection list. To approve a link, select the e-mail message you want to send (if any) from the drop down selection list and then click on the icons. Rejecting a link works the same way, except you will click the icon. To process many links at once, put a check in the box next to the links you want to process. At the bottom of the page select either 'Approve Selected' or 'Reject Selected' from the drop down list of functions. When you do this, an additional drop down selection box will appear that will allow you to select which e-mail message (if any) you want to send. Select the appropriate e-mail message and then press the Execute function to process all of the selected links.

Processing Edited Links

If you have enabled the option to require that link edits be approved before going live in your link directory, you will need to login to the control panel from time-to-time to review the link modifications. To view the edited links, select To Do > Review Edited Links from the control panel menu. This will load the Link Search interface with the appropriate options set to display only links that have been edited. Below the standard link information that is displayed, these links will have additional information that shows you which fields the user has changed.

Each field that has been modified will be listed so you can see the new information that has been provided. Inside the yellow box that contains the new link information you will see the and icons. If you approve the user modifications by clicking the icon, the new data will be assigned to the link and will appear in the link directory at the next cache update. If you reject the user modifications by clicking the icon, the new data will simply be discarded and the existing link information will remain in the directory.

To process several links at once, put a check in the box next to the links you want to process. At the bottom of the page select either 'Approve Edits of Selected' or 'Reject Edits os Selected' from the drop down list of functions. Press the Execute function to process all of the selected links.

Editing a Link

To edit a link you will fist need to locate it through the Directory Browsing or Link Search interfaces. When you locate the link you want to edit, click on the icon to bring up the link editing interface. This interface is nearly identical to the link creation interface, and has the same form fields with the same properties. You can make the changes that you want, and then press the Update Link button to save your changes.

Deleting a Link

To delete a link you will fist need to locate it through the Directory Browsing or Link Search interfaces. When you locate the link you want to delete, click on the icon, which will ask you to confirm the deletion.

You can also delete several links at once by putting a check in the box next to the links you want to delete, selecting the 'Delete Selected' function from the drop down list at the bottom of the page and pressing the Execute button.
