Partner Accounts

Table of Contents
Introduction

Partner accounts allow you to restrict who is allowed to submit galleries at your TGP and/or give certain users special privileges such as being allowed to submit more galleries than a regular submitter. This document will give you an overview of how to add, maintain, and remove partner accounts.

Searching Partners

To search the current partners in your database so you can modify or delete them, access the Partners > Manage Partners function from the control panel menu. This will display a page where you will be able select a number of different search options so you can quickly locate the partner(s) you want to work with. All of the database fields that you can search use the software's Standard Search function. In the table that displays the partners, you will find several functions that you can run on a single partner at a time. Each of those functions are described below.
    Click this icon to suspend the partner account; this will set the partner's status to suspended.
    Click this icon to re-activate a suspended partner; this will set the partner's status back to active.
    Click this icon to edit the partner. A window will appear where you will be able to edit all of the partner settings.
    Click this icon to send an e-mail to the partner. A window will appear where you will enter the e-mail text.
    Click this icon to delete this partner.
In addition to running functions on single partners at a time, you can also run most of these functions on multiple partners at once. At the bottom of the search results table you will find two drop down selection fields and a button labeled Execute. Each of these bulk functions is described below.
    Delete Use this function to bulk delete partners
    E-mail Use this function to bulk e-mail partners. A window will appear where you will be able to enter the e-mail text.
    Suspend Use this function to bulk suspend partners; the partner account status will be changed to suspended
    Activate Use this function to bulk re-activate suspended partners; the partner account status will be changed to active
When you use the bulk functions, you will have the option to specify which partners should be included. The drop down selection list gives you three options:
    Selected Partners Apply this function only to the partners that you selected by checking their associated checkbox in the search results
    All Matching Partners Apply this function to all of the partners that match the search values you have specified
    All Partners Apply this function to all partners in the database
Once you have selected the function you want to run and decided which partners to apply the function to, press the Execute button to run that function. Running functions on a large number of partners can take some time to complete, so be sure to allow the function to run completely before you run other software functions.

Adding a Partner

To add a new partner access the Partners > Add Partner function from the control panel menu. This will display a popup window where you will be able to enter the partner account data that you want to use. Each of the form fields are described below.
    Username The username for this partner
    Password The password for this partner
    Name The name of the partner account holder
    E-mail Address The e-mail address that should be associated with this partner account
    Weight The weight value assigned to this account; galleries submitted through this account will be assigned this weight
    Galleries Per Day The maximum number of galleries this account is allowed to submit per day
    Status The status to assign to this account
    Categories Select the categories this partner account should be allowed to submit galleries to, or select ALL CATEGORIES to allow them to submit to any of your categories.
    Use the above selected categories as an exclusion list
    If you would rather select the categories that the partner is NOT allowed to submit galleries to, check this box. When this is checked, the partner will be able to submit to all of your categories EXCEPT for those listed in the Categories setting.
    Start Date The date and time when this account can begin submitting galleries (must be in YYYY-MM-DD HH:MM:SS format)
    End Date The date and time when this account will no longer be able to submit galleries (must be in YYYY-MM-DD HH:MM:SS format)
    Icons Below the End Date field you will see your currently defined icons (if any). Put a checkbox next to any of the icons that you want to be automatically assigned to galleries submitted by this partner account.
    Allow URL redirection
    By default galleries that redirect by using 30x HTTP status codes are rejected. To allow this partner to submit galleries that redirect, check this box.
    No reciprocal link required
    Check this box to allow this partner to submit galleries that do not contain a reciprocal link
    Auto-approve galleries
    Check this box to have galleries submitted by this account be automatically approved
    No confirmation e-mail required
    Check this box to allow this partner to submit galleries without confirming them through e-mail
    Allow blacklisted items
    Check this box to allow this partner to submit galleries that contain blacklisted data
You will also find all of the user defined fields that you have configured, and will be allow to fill in or select the appropriate values for those fields. Once you have filled in all of the fields as you want them, press the Add Account button. This will add the new account to the database.

Partner Account Status

When partner accounts are created they are assigned a status which determines if the user is able to submit galleries with the account. Each of the available status options are described below.
    Pending New accounts (generally from the partner account request form) that have not yet been approved
    Active Accounts that are active and can submit galleries
    Suspended Accounts that have been suspended by an administrator; the user will not be able to submit galleries
The account status can be changed at any time by editing the account or by using the functions available in the Searching Partners interface.

Reviewing Partner Account Requests

Using the partner.php interface gallery owners will be able to request a partner account at your TGP. Every so often you will need to login to the TGPX control panel to process partner account requests. To view the currently waiting partner accounts access the Partners > Review Partner Requests function from the control panel menu. This will display an interface where you will be able to view and process the partner account requests that have been submitted. The information displayed for each account is identical to when you add an account, with the following two extra items:
    IP Address The IP address of the user that submitted the partner account request
    Rejection If you are rejecting the account request, you can select a rejection e-mail message to send to the submitter. Select NONE to have no e-mail message sent.
To approve a partner account request, first make sure the account information is configured how you want it. You can then put a check in the associated checkbox and scroll to the bottom of the page. Select 'Approve' from the first drop down selection field and 'Selected Partners' from the second drop down field. Press the Execute button to approve the selected accounts. This allows you to approve many accounts at once by putting a check in the checkbox for each account that you want to approve.

All approved accounts will have a password automatically generated for them and the e-mail message defined by the email-partner-added.tpl template will be sent to the user that requested the partner account.

Rejecting partner account requests works exactly like approving them except that you select the Reject option from the first drop down selection field at the bottom of the page. If you select something other than NONE for the Rejection field, the contents of that rejection e-mail message will be sent to the user that requested the partner account.

Processing Inactive Partner Accounts

TGPX includes a couple of functions to process inactive accounts - accounts that have not submitted a gallery in several days. To process inactive partners, access the Partners > Manage Partners function from the control panel menu. Near the top of the page in the header bar that reads "Partner Accounts" click on the icon. This will display a popup window where you will be able to either e-mail or delete inactive accounts.

To send an e-mail message to inactive partner accounts, enter the number of days that the account must have been inactive for and then press the E-mail Accounts button. This will send the contents of the email-partner-inactive.tpl e-mail template to all inactive accounts.

To delete inactive partner accounts, enter the number of days that the account must have been inactive for and then press the Delete Accounts button. This will delete the partner accounts and all of their associated galleries.

Editing a Partner

To edit a partner account you will first need to locate it through the Searching Partners interface. When you locate the account you want to edit, click on the icon to bring up the account editing interface. This interface is nearly identical to the account creation interface, and has the same form fields with the same properties. You can make the changes that you want, and then press the Update Account button to save your changes.

Deleting a Partner

Deleting a partner will also delete all of the galleries associated with that account!

To delete a partner you will first need to locate it through the Searching Partners interface. When you locate the account you want to delete, click on the icon, which will ask you to confirm the deletion.
