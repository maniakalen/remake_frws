Broken and Cheating Link Reports

Introduction

LinkX gives your surfers the ability to report broken and cheating links to you so they can be processed by one of your administrators. By default, on the link details page there is a link that your surfers can use to report a bad link. All they have to do is click that link and enter information on what is wrong with this link. You can then process these reports through the LinkX control panel, which is described in this document.

Viewing and Processing Bad Link Reports

To view the bad link reports that have yet to be processed, access the To Do > Review Reports function from the control panel menu. This will display a page where you will be able to see all of the bad link reports that have been submitted but not yet processed. For each bad link report you will see the information that the user submitted, their IP address, and the date and time the report was submitted. Below each report you will find information about the link that has been reported, including the site URL, title, and description.

In the table that displays the bad link reports, you will find several functions that you can run on a single bad link report. Each of those functions are described below.
    Click this icon to scan the link to see if it is working and/or has any blacklisted items
    Click this icon to see more detailed information about the link that was reported; this will take you to the link search interface
    Click this icon to blacklist the link that was reported and delete it from the database. This will also remove the bad link report from the database. The site URL, e-mail address, and submit IP will be added to the blacklist
    Click this icon to delete the link that was reported from the database, but not blacklist it. This will also remove the bad link report from the database.
    Click this icon to delete this bad link report and do nothing with the link that was reported
You can also run a function on multiple bad link reports by putting a check in the box next to the reports you want, selecting the function to run from the drop down selection field at the bottom of the page, and pressing the Execute button. Each of the available functions are described below.
    Ignore Selected This will delete the selected link reports without taking any action against the link that was reported
    Delete Selected This will delete the link that was reported and remove the report from the database.
    Blacklist Selected This will blacklist the link that was reported and remove the report from the database.
    The site URL, e-mail address, and submit IP will be added to the blacklist
