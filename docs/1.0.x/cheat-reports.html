Broken and Cheating Link Reports

Table of Contents
Introduction

TGPX gives your surfers the ability to report broken and cheating galleries to you so they can be processed by one of your administrators. If you choose to add a link to the cheat reporting interface, your site users will be able to click that link and report broken galleries to you. You can then process these reports through the TGPX control panel, which is described in this document.

Viewing and Processing Bad Gallery Reports

To view the bad gallery reports that have yet to be processed, access the To Do > Review Reports function from the control panel menu. This will display a page where you will be able to see all of the bad gallery reports that have been submitted but not yet processed. For each bad gallery report you will see the gallery URL of the reported gallery, the information that the user submitted, their IP address, and the date and time the report was submitted.

In the table that displays the bad gallery reports, you will find several functions that you can run on a single bad gallery report. Each of those functions are described below.
    Click this icon to scan the gallery to see if it is working and/or has any blacklisted items
    Click this icon to blacklist the gallery that was reported and delete it from the database. This will also remove the bad gallery report from the database. The hostname from the gallery URL, e-mail address, and submitter IP will be added to the blacklist
    Click this icon to delete the gallery that was reported from the database, but not blacklist it. This will also remove the bad gallery report from the database.
    Click this icon to delete this bad gallery report and do nothing with the gallery that was reported
You can also run a function on multiple bad gallery reports by putting a check in the box next to the reports you want, selecting the function to run from the drop down selection field at the bottom of the page, and pressing the Execute button. Each of the available functions are described below.
    Ignore This will delete the selected gallery reports without taking any action against the gallery that was reported
    Delete This will delete the gallery that was reported and remove the report from the database.
    Blacklist This will blacklist the gallery that was reported and remove the report from the database.
    The hostname of the gallery URL, e-mail address, and submitter IP will be added to the blacklist
