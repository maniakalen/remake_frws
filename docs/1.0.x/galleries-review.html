Reviewing New Galleries

Table of Contents
Introduction

TGPX has a built in tool that will allow you to quickly review newly submitted galleries. While reviewing galleries you will be able to see the gallery itself, make changes to the gallery information, and approve or reject the gallery submission. This document will provide information on the gallery review utility.

The Gallery Review Interface

To use the gallery review interface you will need to setup your popup blocker to allow popup windows from the site where TGPX is installed.

To begin reviewing galleries, access the Galleries > Review New Galleries function from the control panel menu. This will display two popup windows, one that contains the gallery information that you can edit and one that shows the gallery itself. The popup window that displays the gallery information will look something like this:


  1. This is the gallery data which you can edit if any changes are required
  2. This shows the number of galleries left to review along with the number of galleries you have skipped in parenthesis
  3. These icons allow you to process the current gallery, skip a gallery, and set options
  4. This section shows the preview thumbnail(s) for the gallery (if any) and allows you to modify or delete them
Gallery Review Functions

Each of the icons in section C of the gallery review interface correspond to a function that you can run. Each of those functions are described below.
    Click this icon to approve the gallery you are currently viewing. You will be asked to confirm the approval before it is submitted. When the gallery is approved, any changes that you have made to the gallery data will be recorded in the database. If you have the option enabled to notify submitters when their gallery is approved, the approval e-mail will be sent.
    Click this icon to reject the gallery you are currently viewing. A selection field will slide into view that will allow you to select what rejection e-mail message you want to send. To reject the gallery without sending an e-mail message, select NONE from the drop down selection list. Once you have selected the e-mail you want to send (or NONE) click on the icon to complete the rejection.
    Click this icon to delete the gallery you are currently viewing. This is the same as rejecting the gallery except that no rejection e-mail will be sent. You can use this as a shortcut to reject galleries when you do not want to select a rejection e-mail to send.
    Click this icon if you want to blacklist and delete the gallery you are currently viewing. When you click this icon several input fields will slide into view allowing you to enter the data that you want to blacklist for this gallery along with a reason for blacklisting the data. Once you have filled in the values as you like, click on the icon to actually blacklist and delete the gallery.
    Click this icon to skip the current gallery you are viewing and move on to the next available gallery
    Click this icon to set options that control which galleries you are reviewing and what order they appear in. When you click this icon the search interface will slide into view. This interface will allow you to select which galleries you want to review by selecting the type, format, sponsor and/or category you want to work with. You can also select how the galleries should be ordered while you review them. Once you have the settings you want to use, press the Save button which will load the first gallery that meets the criteria you have set.
Keyboard Shortcuts

The gallery review interface contains a few keyboard shortcuts that you can use to approve, reject, delete, or move onto the next gallery. These shortcuts will work only when one of the form fields does not have the input focus.
    ctrl-q Press the 'ctrl' and 'q' keys on your keyboard to remove focus from the forum input fields so you can use the other shortcuts
    a Press the 'a' key on your keyboard to approve the current gallery
    r Press the 'r' key on your keyboard to reject the current gallery
    d Press the 'd' key on your keyboard to delete the current gallery
    right arrow (->) Press the right arrow key on your keyboard to skip the current gallery and move onto the next


Gallery Data

Most of the gallery settings can be edited through this interface by editing the values in section A. Each of the values that you can edit are described below.
    URL The full URL to the gallery
    Description The description of the gallery
    Keywords A space separated list of keywords that describe the gallery. Keywords must be at least 4 characters long
    Tags A space separated list of tags for this gallery. Tags can be used for grouping galleries.
    Sponsor The sponsor that this gallery comes from
    Categories The categories that the gallery fits into. Multiple categories can be selected by clicking the icon. You can remove categories by clicking on the icon.
    If you have configured any icons, you can assign them to a gallery by clicking on this icon. A selection box will appear where you can put checks in the checkboxes next to any of the icons you want to assign to this gallery.
    Weight The weight value to assign to this gallery. The weight value can be any whole number from -2147483648 to 2147483647. Gallery weight values only come into play if you are using the weight sorting option in the order or reorder settings of the {galleries} template command
    Scan
    Uncheck this option if you do not want the gallery scanner to scan this gallery
    Thumb
    Uncheck this option if you do not want the gallery scanner to be allowed to create a preview thumbnail for this gallery
    Thumbnails The number of thumbnails that appear on the gallery (i.e. small images that link to the full sized image or movie)
    Scheduled The date and time that this gallery will become eligible for display on your TGP pages. Note this does not guarantee that the gallery will be displayed on this date, it simply becomes eligible for selection on this date.
    This must be in YYYY-MM-DD HH:MM:SS format.
    Delete The date and time that this gallery should be deleted from the database. This must be in YYYY-MM-DD HH:MM:SS format.
    Type The type of gallery, either permanent or submitted
    Format The format of this gallery, either movies or pictures
    Nickname The name of the gallery submitter
    Submit Info This will display the submitter's e-mail address, IP address, and the date and time the gallery was submitted
If you have configured any user defined gallery fields, they will appear below the Categories field in the gallery review interface. All of these settings can be edited before you approve a gallery; the changes will be recorded in the database when the gallery is approved.

Preview Thumbnails

If the gallery you are reviewing has any preview thumbnails, those will appear in the gallery review interface in section B. The drop down selection list will show you how many preview thumbnails have been created for this gallery, and each will be listed by it's dimensions. To view one of the other preview thumbnails for the gallery, simply select it from the drop down list.

To create a new preview thumbnail for the gallery click on the icon. This will display up a new popup window that will allow you to crop the image of your choice from the gallery.

To delete a preview thumbnail, first select it from the drop down selection field and then click on the icon.
